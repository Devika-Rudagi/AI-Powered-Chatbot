# AI-Powered Chatbot Web Application

A professional full-stack conversational application built with a **React** frontend and a **.NET 8/9** backend, integrating the **OpenAI API** for intelligent responses.

## 🚀 Features
- **Real-time Chat:** Interactive UI with state management for a seamless experience.
- **Secure Backend:** .NET Core API acts as a proxy to keep OpenAI keys hidden from the client.
- **Structured JSON Handling:** Robust communication between client and server using JSON.
- **Modern Tech Stack:** Utilizes Axios for HTTP requests and the latest OpenAI .NET SDK.

## 🛠️ Tech Stack
- **Frontend:** React, CSS3, Axios
- **Backend:** .NET Core API, C#
- **AI:** OpenAI GPT-4o Integration

## ⚙️ Setup
1. **Backend:**
   - Navigate to `/server`.
   - Update `appsettings.json` with your OpenAI API Key.
   - Run `dotnet run`.
2. **Frontend:**
   - Navigate to `/client`.
   - Run `npm install` and `npm start`.
